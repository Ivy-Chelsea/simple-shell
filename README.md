<h1>Simple Shell</h1>
<h2>Description</h2>
<p>This shell project done by Ivy and Anna will demonstrate how a simple UNIX command interpreter was created. </p>
<h2>Compiling requirements:</h2>
<p>gcc -Wall -Werror -Wextra -pedantic -std=gnu89</p>
